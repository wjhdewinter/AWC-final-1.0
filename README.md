# AWC Warehouse Tool — Versie 1.0

Definitieve schone ZIP voor GitHub Pages of lokaal gebruik.

## Belangrijkste functies
- Personeel check met keuze: alleen afwezigen of volledige lijst versturen
- Afwezigen worden apart vermeld in PDF, e-mail en WhatsApp
- E-mail en PDF zijn zakelijk zonder emoji’s
- WhatsApp blijft informeel en duidelijk
- Order wegen met colli, pallets, artikelnummers, aantallen, afmetingen en bestanden/foto’s
- Schade en manco registratie met mail/PDF/Excel
- Warehouse check per locatie met passende vragen
- Nederlands/Engels taalkeuze wordt onthouden
- Update-knop tegen oude browsercache
