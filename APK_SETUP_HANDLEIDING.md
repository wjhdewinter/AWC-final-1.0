# AWC Warehouse Tool - APK setup

Deze map is klaar om via PWABuilder als Android APK te bouwen.

## 1. Upload naar GitHub Pages
Upload alle bestanden uit deze map naar je GitHub Pages repo.
Belangrijk: `index.html`, `manifest.json` en `sw.js` moeten in de hoofdmap staan.

## 2. Controleer de live link
Open je live URL, bijvoorbeeld:

`https://wjhdewinter.github.io/AWC-final-1.0/`

Controleer dat de app opent en dat bovenin alleen `Versie 1.0` staat.

## 3. APK maken met PWABuilder
1. Ga naar PWABuilder.
2. Vul je live GitHub Pages link in.
3. Klik op Start.
4. Kies Android package.
5. Download de APK/AAB.
6. Installeer de APK op je Android telefoon.

## 4. Aanbevolen Android instellingen
- App name: AWC Warehouse Tool
- Short name: AWC Tool
- Package ID: com.amsterdamwarehouse.awctool
- Display mode: standalone
- Orientation: portrait
- Theme color: #111111
- Background color: #111111

## 5. Belangrijk bij updates
Als je later een nieuwe APK maakt, verhoog dan de versie in PWABuilder.
Bij GitHub Pages updates: druk in de app op Update of wis de browsercache als je oude data blijft zien.

## 6. Inbegrepen iconen
Deze setup bevat PWA/APK iconen in meerdere formaten:
48, 72, 96, 128, 144, 152, 192, 384 en 512 px, plus maskable icons.
